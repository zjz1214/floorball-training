# 传球训练

::: tip
传球训练是射门的基础，很多动作是相似的，要优先掌握
传球可以是拖的也可以是敲击的，后者会更快，前者会更稳定
:::
1. **正面快速传球**：要点是传球时的主动快速发力；球在身前，不需要向后缓冲很多；控制杆子角度不过分上扬，身体将球杆整体送出去；拍面略内扣使球不跳跃；

双人移动快速传球：这个是比赛中常用的动作，通过两个人的快速传导实现对防守人员的调动
2. **身后护球的传球：****传球应当是主动发力传出去的，而不是靠身体向前冲的惯性甩出去的**

4. **反手传球练习**
    1. 反手击打
    2. 反手推出去
5. **robin4人传球**要点：
    1. 提高传球准度和速度
    2. 减少球在拍头上的停滞时间
    3. 练习的时候传球可以尽量快，实际中应当视距离控制球速
    4. 接球人双手持杆，亮起拍面表明接球位置
