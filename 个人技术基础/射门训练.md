# 射门训练

## 要点

1. 射门的核心原理：提高出球那一下杆子的速度。
   - 适当压杆，但是在阻力过大或过小的地面上难以压杆
   - 下手要主动发力往上抬，可以不拿杆子模拟一下下手特别使劲发力往上甩，一般来讲会觉得别扭。这是因为自然的动作中很少会有这种快速抬胳膊的动作，肌肉力量和肩关节灵活度是不足的。
2. 进阶：在7-8米内的距离，射门准度的重要性大于射门的力度。所以射门的时候轻巧一些，少压杆少使劲。少使劲后会发现反而发力变的更加集中射门速度也不会下降。
3. 有效的射门的分区主要是两个上角两个腋下以及两个底角，根据具体情形选择。
4. 掌握全面的射门技术，在不同的场景下灵活使用射门技术。
5. 离门近的时候多做快速的击打射门，而不是需要更长前摇的拖射。

#### outsweeper shot

护球转体打门，身体的起始朝向是水平的，球放置在身侧。

1. 远端脚支撑，踩远踩实，球的轨迹是一条弧线（趋近于直角，先向前再折向垂直方向）
2. 球的加速（集中发力）过程应当在将球基本上扭过来之后（在朝向门的方向）再进行，不要再画弧的过程中加速。

### 动态的拖射（进阶1）

#### 向前移动的

<video src="/videos/LLJvbc6vlopwR6xqdVic2X0YnJb.mp4" controls autoplay loop muted playsinline></video>

<video src="/videos/A5GBbnVnHoy9e8xlYf8cpxmZn7f.mp4" controls autoplay loop muted playsinline></video>

1. 起步的时候为了将球调整到身后的位置使用了拍头拉球的方式（另一种就是反拍给他拉回来，前者会更舒适）
2. 球没动，人的相对位置向前，从而实现准备过程中球的后置及上手带动身体的牵拉转体（所有拖射这一步很重要）

![IBmwbRTTYojmFQxgKYxcr3m0nse](/IBmwbRTTYojmFQxgKYxcr3m0nse.png)

3. 头应当在球完成后置时就一直看向球门，需要有一定球感保持球不掉落
4. 支撑脚向外大步迈出踩实，（踩支撑脚的前一步也可以有意识的踩实一点）。身体顺势转体并向外飞出（另一条腿会甩过来）射门前的前两步脚步一定要迈得大一些

![Qv1ybI1dUo5fJaxQ06OcWuULnOc](/Qv1ybI1dUo5fJaxQ06OcWuULnOc.png)

### 接传球的拖射（进阶2）

#### 接到前方来球的 sweeper shot

<video src="/videos/YtyrbZMhyojRahx32cAcZEKdnte.mp4" controls autoplay loop muted playsinline></video>

<video src="/videos/RCUGbqkBmoC5imxXhhFcj5jDn8f.mp4" controls autoplay loop muted playsinline></video>

1. 类似的，身前接球，拍头先水平停下球，然后拍头微向后拉，球保持停止或微往后动。然后身体向前完成蓄力。下手只在打门蓄力阶段握实
2. 身后的带球这一段，拍面可以稍微往外翻一点，让球滑动到拍头，这样不容易掉球，因为大部分时候球是从拍根部滑出去的
3. 同样观察到上手主动往后
