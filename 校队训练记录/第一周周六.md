# 第一周周六训练

| 序号 | 动作 | 说明 | 组数 | 时间 |
|------|------|------|------|------|
| 1 | 热身 | 绕场慢跑2圈，列队热身 | | 10mins |
| 2 | 手传球小游戏 | 分两队，不用球杆，手传球将球传进球门。<br/>规则：<br/>— 拿到球后不能动，只能传球，不能回传给传给你的人<br/>— 门后有一个点，射门前必须将球传给门后踩在点上的人再传出来后射门，同样不能回传<br/>— 球不能落地，如果球被断下/没接住交换球权 | | 10mins |
| 3 | Robin 4人传球远距离 | 距离加长，球速加快<br/>![Robin传球](/training_log_img01.png) | | 10mins |
| 4 | 传射训练 | 中场两边站两队，给对面传球，对面接球向前斜插打门，两边交替。在反手侧做腕射，在正手侧做拖射<br/>![传射训练](/training_log_img02.png) | | 10mins |
| 5 | 追逐跑训练 | 前面一个带球的往前冲，后面一个追。后面先将球传给前面的人，然后开始追逐。跑的人到对面后打门结束，追的人到对面接下一个人传球开始跑……<br/>![追逐跑](/training_log_img03.png) | | 10mins |
| 6 | 4v4 训练赛 | | | 剩余时间 |
