# 个人战术

球员的个人特征由技术和体能以及比赛感共同构成。比赛感包括理解比赛、阅读比赛和决策。阅读当前的比赛情境并预判下一步动作帮助球员根据所处位置做出最佳决策。个人战术借助**比赛情境角色**学习最佳。

## 比赛情境角色

**持球人**有三个选项。首要目标是得分，因此射门。第二个选项是传给得分机会更好的球员。第三个选项是带球到更好的射门或传球位置。这需要反应速度和阅读比赛的能力。虽然持球人有责任做出决策，无球球员也有责任通过遮挡、创造空间和传球线路等方式协助他/她。

在比赛中，所有角色和职责相互作用。它们都需要球员评估空间、速度、方向和时机的能力。不同比赛情境角色的职责以及球员在某角色中应具备的技术技能如下表所示。但必须理解，比赛中角色转换极快，球员仅在极短时间内处于某一角色。对青少年球员，不应如表中所示强调这些个人战术。这可能以完全混乱收场，所以角色和职责应在比赛感发展的过程中逐渐引入。

![比赛情境角色](/iff-tactics-roles.png)

| 角色 | 球权 | 职责 | 所需技术技能 |
|------|:----:|------|-------------|
| **持球人（进攻）** | 有 | 接球/获球权、护球、射门、传球、通过带球移动到更好的射门或传球位置 | 移动、球杆操控、假动作、传球、射门、护球、1v1 |
| **防守持球人** | 无 | 将持球人引向不利区域、剥夺空间、抢球、封堵传球线路、封堵射门、准备进攻、积极人盯人和延缓对手 | 变向移动、封堵传球线路和射门、球杆阻挠、拦截、1v1 |
| **无球进攻球员** | 无 | 通过移动创造传球线路、保持与持球人的近距离、为持球人创造空间、摆脱接传球或射门、遮挡守门员、准备防守 | 移动、遮挡、假动作、接球、1v1 |
| **防守无球球员** | 无 | 盯人、封堵传球线路、准备进攻、保持对球和对手的视觉接触、保持在对手和己方球门之间 | 变向移动、封堵传球线路、盯人、球杆阻挠、1v1 |

通过将局面从 2v2 转化为 2v1 甚至 2v0 来以多打少的能力也至关重要。通过创造区域强侧（小区域内 2v1、3v2 或反过来），帮助防守队从对手处抢球，进攻方面则始终创造更好的得分机会。

教授服务于团队战术的个人战术是重要课题，尤其在减少比赛情境中的自私行为时。应激励球员思考并选择何时自私，何时传球或帮助队友。

下一阶段因此是教球员如何阅读情境并以最佳选项做出反应。通过在练习中增加更多球员，迫使球员根据情境快速决策。个人技能应掌握到球员在比赛中无需思考技术执行的程度。这当然是理想状态，因为球员——尤其年轻球员——几乎从不专注于执行，因此技术技能应持续练习。

## 一级

几乎所有前述练习都包含个人战术技能和决策。所有选项多于一个且球员被迫做出决策的练习，都培养球员阅读比赛的能力。

一级中选项数量应最少，练习应简单。对年轻球员，简单意味着专注于显而易见的事物——决策；对更高级的球员，练习可视为热身练习，他们可例如专注于射门。教练始终有权选择想向球员强调什么技能。3v0 因为选项数量多而要求高，3v2 因为时间和空间减少而要求高。

### 练习 40 {#drill-40}

**训练目标：**
- 练习决策：传球还是射门
- 为守门员热身

**组织安排：**
- 球员在角球位置带球排成两列
- 可在场地两端同时执行

**执行步骤：**
- A1 开始传一个有力斜线长传给 A2
- A2 在移动中接球
- 两名球员向中线和中心跑动，在中心交叉
- A2 可在交叉时回传给 A1 或给回敲传球
- 持球人可射门或传球
- 执行后换线

**变化形式：**
- 第一次传球可从球门后或由守门员传出
- 也可使用整个场地，从两端交替执行练习

**要点提示：**
- 球员应随时准备传和接
- 无球球员如未接到传球不应沮丧（强调射门）
- 执行应快速，不超过 4-5 次传球
- 如使用整场执行，两端执行可重叠：前一组过中线时下一组开始
- 此情况下球员应向得分区域移动以避免碰撞

### 练习 41 {#drill-41}

**训练目标：**
- 练习 1v1 情境中的决策
- 球员被迫在带球和射门之间选择

**组织安排：**
- 场地纵向一分为二
- 球员在场地两侧各排两列
- 一列在角球位置，一列在中点
- 球在角球位置

**执行步骤：**
- A1 从角球位置传球给 A2 开始
- A2 回传给 A1，A1 控球，在场地另一端进行 1v1
- 另一端的球员执行相同动作
- 执行后 A1 变为 A2，A2 变为 B1

**变化形式：**
- 也可只使用半场执行：A1 接球后在中线前转身
- 球员较少或只有一个守门员时使用此变化

**要点提示：**
- 重点应在做决策：是带球并尝试假动作将防守球员引向另一方向，还是在防守球员有距离时射门
- 技术执行应足够熟练，使持球人无论决定射门还是带球都能获得得分机会

### 练习 42 {#drill-42}

**训练目标：**
- 练习 3v1 情境中的决策
- 选项因球员数量而增加，但大量时间和空间应使球员决策更容易

**组织安排：**
- 球员在半场四个角各排一列
- 球在一个角

**执行步骤：**
- B1 从球门后传球给 A1，随后进入球门区前的防守角色
- A1 传球给 A2 并跟随传球
- A2 控球，同时 A3 从各自侧出发向 A2 方向跑动
- A2 给 A3 回敲传球
- 之后他们进攻，B1 防守
- 执行后各线按顺时针轮换

**变化形式：**
- 也可先无防守球员执行，以及加两名防守球员
- 也可使用整场

**要点提示：**
- 执行应快速，始终以射门结束
- 教练应让球员先做自己的决策，然后在给予反馈时询问决策理由

### 练习 43 {#drill-43}

**训练目标：**
- 进攻球员练习决策：射门还是传球
- 练习得分和球门前的 1v1 情境

**组织安排：**
- A1 在球门前作为进攻球员
- B1 在球门前防守 A1
- A2 和 A3 在各自侧翼
- 其余球员在中点带球排成一列

**执行步骤：**
- A4 传球给 A1，A1 须做出决策：如有足够空间则转身射门（选项 1），或传给 A2 或 A3（选项 2）
- A2 和 A3 应移动并通过创造传球线路协助 A1，且随时准备快速射门
- B1 应尝试阻挠 A1 做决策
- 如 A1 传给 A2 或 A3，B1 应尝试封堵射门
- 如 A1 传给 A2，A3 应尝试遮挡守门员
- 执行后 A1 变为 A2，A2 变为 A3，A3 变为 A4，A4 变为 B1

**变化形式：**
- 传球也可为空中传球，使接球球员技术挑战更大
- 也可规定 A2 或 A3 必须去遮挡守门员，使 A1 只有一个传球机会
- 这也迫使无球球员做出决策

**要点提示：**
- A1 应向来球方向移动以拉开与 B1 的距离，可转身射门
- 如 B1 能够缩小间距，A1 应保持抬头观察传给谁
- 接球控球应快，随后的动作同样快
- A2 和 A3 应准备好接球射门或遮挡

## 二级

通过改变空间和时间量来改变练习的要求水平。迫使球员在紧逼情境中快速决策，培养比赛中同样能做到的能力。通过不断改变球员数量创造情境，这将自动减少时间和空间，从而培养球员的决策能力。

下一级中，选项因球员数量而增加。练习多为连续性的，因此球员须随时准备阅读和反应。这些练习可能不适合初学者，但球员对比赛情境角色有些意识后应尽快引入。

### 练习 44 {#drill-44}

**训练目标：**
- 培养理解比赛情境角色以及每个角色中球员不同选项的能力
- 练习移动、传球，以及通过封堵传球线路和人盯人防守阅读对手

**组织安排：**
- 场地分为四个区域
- 每区进行 2v2 比赛
- 见练习 38

**执行步骤：**
- 教练发出 45-60 秒轮换信号，之后球员换对手
- 目标是无特殊规则进行 2v2 比赛

**变化形式：**
- 教练可添加规则以强调特定技能
- 强调传球：球员只能触球两次，然后必须传给队友
- 缩小区域：强调在紧逼情境中护球

**要点提示：**
- 同时跟进四场比赛对教练可能困难，这使球员有责任在有规则时严格遵守
- 球员应保持抬头观察队友和对手位置

![练习44场地示意图](/iff-drill-44.png)

### 练习 45 {#drill-45}

**训练目标：**
- 练习 2v2 情境中的决策，所有比赛情境角色均存在并不断变化
- 从防守转进攻或反之的转换——取决于起始情境

**组织安排：**
- 球员排成四列，两个球门两侧各两列
- 球门可拉近使场地变小，或球门可置于场地一端

**执行步骤：**
1. 守门员传球给第一对进攻球员（A1 和 A2）。A1 和 A2 进行 2v0 进攻
2. 当球被守门员扑住或出界，守门员传球另一球给 B1 或 B2 开始进攻
3. A1 和 A2 从进攻快速转换为防守
4. 防守 B1 和 B2 后，A1 和 A2 回到己方球门后，另一对 A3 和 A4 开始攻击 B1 和 B2
5. 如 A1 或 A2 成功从 B1 和 B2 处抢断，可立即传球给 A3 或 A4
6. 练习持续至教练叫停

**变化形式：**
- 强调防守转进攻：练习以 2v2 开始，其中 A1 和 A2 攻击 B1 和 B2；攻击后 B1 和 B2 攻击已在己方球门前站位防守的 A3 和 A4

**要点提示：**
- 初学者更容易告诉他们应先攻击还是先防守
- 等待轮次的球员应准备好就位
- 防守球员应在中线迎上进攻
- 进攻中球员应向中线方向移动以在移动中接球
- 守门员应在做出扑救或球出界后立即准备传球

## 三级

三级的练习均以每方超过两名球员执行。核心是球员意识到所有其他球员及其位置，以及自己的角色。这需要高水平的理解和阅读比赛的能力。对青少年球员或初学者，不应期望他们能够阅读比赛并对每个情境做出相应反应，因此目标应根据球员水平来解释。

执行不同团队战术包含在练习中，如开球、攻防转换和防守战术。最重要的是球员在按某战术体系比赛时须执行的个人技能。

### 练习 46 {#drill-46}

**训练目标：**
- 练习为获得得分情境的决策
- 练习所有不同比赛情境角色并定义角色中的选项

**组织安排：**
- 球员在场地一端球门线后排列成五列
- 球在同一端

**执行步骤：**
1. A1 开始攻击守门员
2. 射门后 A1 转身防守 A2 的进攻——A2 在 A1 过中线后立即出发
3. A2 过中线后，B1、B2 和 B3 对 A1 和 A2 发起进攻
4. 执行后各线按顺时针轮换

**变化形式：**
- 也可只在一端执行
- 球员须绕过中线处放置的标志桶

**要点提示：**
- 球员应按正确顺序轮换以保持各线人数均衡
- 球员应准备好快速决策，因为他们必须对接下来的情境迅速做出反应
- 球员应在执行后立即清理场地，为下一批球员留出空间
- 球员应意识到队友和对手的位置
- 在 1v1 情境中，应鼓励持球人挑战对手

### 练习 47 {#drill-47}

**训练目标：**
- 培养球员通过以多打少阅读比赛情境的能力
- 练习从防守到进攻的快速转换及进攻准备
- 通过增加球员数量（从而增加选项数量）至 3v3 情境
- 通过减少时间量培养球员快速决策的能力

**组织安排：**
- 球员在中线板墙处排成两列，面向各自球门
- 两名球员 B1 和 B2 在己方球门前准备防守
- 球在球门内或球门框上方

**执行步骤：**

**阶段 1：** A1 和 A2 接守门员传球，开始对 B1 和 B2 的进攻。A1 和 A2 过中线后，第三名防守球员 B3 回追防守，准备接 B1、B2 或守门员的开球传球。

**阶段 2：** 当球被扑住、出界或防守球员（此例为 B1 或 B2）获得球权，B1、B2 和 B3 发起另一次进攻。A3 和 A4 此时已在己方球门前站位防守进攻。A1 和 A2 回到场地中路的起始线。

**阶段 3：** 进攻球员过中线后，回追球员 A5 协助防守球员；如防守球员获得球权，A5 准备进攻。

**变化形式：**
- 可先以 2v2 进行练习使球员更容易理解
- 球员理解轮换后尽快加入第三名球员

**要点提示：**
- 防守球员应准备好防守，一旦获得球权应有进攻准备
- 球员在每个角色中有各种选项，应鼓励他们发挥创造力
- 回追球员应准备好接开球传球，或在防守球员获球权后立即冲刺至槽区
- 应通过快速转换和在进攻区域创造强侧局面来以多打少
- 强侧局面由回追球员创造

### 练习 48 {#drill-48}

**训练目标：**
- 练习 2v3 情境中的决策
- 发展个人战术技能：
  - 防守球员练习前场逼抢以获得球权
  - 进攻球员（持球方）练习通过保持己方防守区域球权来防守，同时练习从防守到进攻的转换
- 也基于球队选定的体系练习团队战术

**组织安排：**
- 球员在场地一端球门后排成五列
- 三名防守球员（B1, B2, B3）和两名进攻球员（A1, A2）

**执行步骤：**
- 守门员向角球方向扔球
- A1 或 A2 首先跑动以获得球权
- B1、B2 和 B3 紧随 A1 和 A2 跑动，通过在指定位置站位以获得球权：
  - 如球队选定使用箭头逼抢球员（1-2-2 阵型），仅有 B1 覆盖持球人
  - 如球队选定在角球位置包夹持球人，B1 和 B2 紧随持球人，B3 覆盖无球球员
- 当然，情境取决于球员的时机把握，因此他们需要有一定的阅读情境能力
- 如 A1 和 A2 成功保持球权，他们可进行一次反击
- 如前场逼抢球员获得球权，他们将尝试得分，而 A1 和 A2 站位防守
- 练习持续至教练吹哨

**变化形式：**
- 可通过将另一球门和线移近另一端使场地缩小，但不能近于中线（如图所示）
- 也可增加第三名进攻球员 A3 协助 A1 和 A2（当及如果他们获得球权）

**要点提示：**
- 球员应准备好防守
- 球员应按照球队在前场逼抢情境中使用的战术比赛
- 应考虑交替情境：如 B1 跟随持球人，B2 和 B3 应知道如何站位以及应盯人还是覆盖区域
- A1 和 A2 也应意识到队友位置；例如如 A1 去拿球，A2 应足够靠近以便接传球并准备进攻
